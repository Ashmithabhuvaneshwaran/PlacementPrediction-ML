⚡ QUICK TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

ISSUE 1: "python: command not found"
SOLUTION:
  • Python not installed or not in PATH
  • Install Python from: python.org
  • Make sure to check "Add Python to PATH" during installation

ISSUE 2: "ModuleNotFoundError: No module named 'flask'"
SOLUTION:
  Run: pip install -r requirements.txt
  Then: python app.py

ISSUE 3: "Port 5000 already in use"
SOLUTION:
  • Another application is using port 5000
  • Option 1: Close the other application
  • Option 2: Edit app.py, change last line from:
              app.run(debug=True)
              to:
              app.run(debug=True, port=5001)

ISSUE 4: "Connection refused" when opening browser
SOLUTION:
  • Wait a few seconds for Flask to start
  • Check terminal for error messages
  • Try: http://localhost:5000 instead of http://127.0.0.1:5000

ISSUE 5: "AttributeError: 'DecisionTreeClassifier' object has no attribute..."
SOLUTION:
  Run: python retrain_models.py
  This retrains the models with current sklearn version

ISSUE 6: "ValueError: X has 12 features, but RandomForestClassifier..."
SOLUTION:
  Run: python retrain_models.py
  This fixes feature mismatch

📊 WHAT THE MODELS DO
════════════════════════════════════════════════════════════════════════════════

PLACEMENT PREDICTION MODEL:
  Input:  Student profile (11 features)
  Output: "Placed" or "Not Placed"
  Uses:   Random Forest Classifier
  Accuracy: 93.27%
  
  Features used:
    1. CGPA (GPA score)
    2. Major Projects count
    3. Workshops/Certifications count
    4. Mini Projects count
    5. Skills count
    6. Communication rating
    7. Internship experience (Yes/No)
    8. Hackathon participation (Yes/No)
    9. 12th Grade percentage
    10. 10th Grade percentage
    11. Academic backlogs


SALARY PREDICTION MODEL:
  Input:  Student profile (11 features) + Placement status
  Output: Expected annual salary in INR
  Uses:   Random Forest Classifier
  Accuracy: 67.93%
  
  Predicts salary only if student is "Placed"
  Example: ₹400,000 - ₹700,000 per annum
