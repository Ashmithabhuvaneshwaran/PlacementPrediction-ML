🆘 QUICK TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────────

Issue: "python command not found"
  → Install Python from python.org
  → Add to PATH during installation

Issue: "No module named flask"
  → Run: pip install -r requirements.txt

Issue: "Port 5000 already in use"
  → Change port in app.py to 5001
  → Or close the application using port 5000

Issue: "Connection refused"
  → Wait 3 seconds for Flask to start
  → Try: http://localhost:5000

Issue: Models error
  → Run: python retrain_models.py

🎓 WHAT THE APPLICATION DOES
───────────────────────────────────────────────────────────────────────────────

INPUT:
  • Student's academic profile (11 features)
  • CGPA, projects, skills, internships, grades, etc.

PROCESSING:
  • Runs through trained Random Forest models
  • Model 1: Determines if student will be placed
  • Model 2: Predicts salary if placed

OUTPUT:
  • Placement Status: "Placed" or "Not Placed"
  • Expected Salary: If placed, annual salary in INR
  • Recommendations: If not placed, improvement suggestions
